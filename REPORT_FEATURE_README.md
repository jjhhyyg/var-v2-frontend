# VAR 熔池分析报告生成功能

## 功能概述

本功能为 VAR 熔池分析系统新增了完整的报告生成与导出功能，支持将任务分析结果导出为 **HTML** 和 **PDF** 格式的专业报告。

## 主要特性

### 📊 报告内容

生成的报告包含以下完整信息：

#### 1. 任务基本信息

- **任务名称**
- **任务ID**
- **任务创建时间**
- **任务开始时间**
- **任务结束时间**
- **任务耗时**（自动计算）

#### 2. 视频信息

- **视频时长**（秒）
- **视频帧率**（FPS，自动计算）
- **总帧数**
- **任务状态**（正常完成/超时完成）

#### 3. 动态参数统计

包含每个动态参数的详细统计：

- **平均亮度** - 熔池亮度的平均值
- **平均面积** - 熔池面积的平均值（像素）
- **平均周长** - 熔池周长的平均值（像素）
- **平均圆度** - 熔池圆度的平均值（0-1范围）

> **圆度计算公式**: `圆度 = 4π × 面积 / 周长²`
>
> 圆度值范围为 0-1，值越接近 1 表示形状越接近完美圆形。

每个参数都会显示：

- 整个视频期间的变化趋势
- 平均值作为参考基线

#### 4. 异常事件统计

按事件类型分组展示所有检测到的异常事件：

- **粘连物形成** - 电极上形成粘连物
- **粘连物掉落** - 粘连物脱落，包含掉落位置信息
  - 掉落进熔池中
  - 被结晶器捕获
- **锭冠脱落** - 锭冠掉落事件
- **熔池未到边** - 熔池未达到边缘
- **熔池到边** - 熔池到达边缘
- **熔池溢出** - 熔池溢出事件

每个事件包含：

- 发生时段（开始时间 - 结束时间）
- 持续时长
- 特殊位置信息（如粘连物掉落位置）

#### 5. 追踪物体统计

详细列出所有被追踪的物体：

- 物体ID
- 物体类别
- 首次出现时间和帧号
- 最后出现时间和帧号
- 持续时长

## 使用方法

### 在任务详情页导出报告

1. 打开任务详情页面
2. 等待任务分析完成（状态为"已完成"或"已完成(超时)"）
3. 在页面右上角找到报告导出按钮区域
4. 点击 **"导出 HTML 报告"** 或 **"导出 PDF 报告"**
5. 浏览器会自动下载生成的报告文件

### 报告预览

如果需要在导出前预览报告内容，可以使用 `ReportCharts` 组件查看动态参数的可视化图表。

## 技术实现

### 纯前端实现

报告生成完全在浏览器端完成，无需后端支持：

- **HTML 导出**: 直接生成 HTML 文件并触发浏览器下载
- **PDF 导出**: 使用 `html2canvas` 将 HTML 转换为图片，再用 `jspdf` 生成 PDF

### 依赖库

```json
{
  "jspdf": "^3.0.3",
  "html2canvas": "^1.4.1",
  "chart.js": "^4.5.0"
}
```

### 核心组件

#### 1. `useReportGenerator.ts` - 报告生成核心逻辑

提供以下功能：

- 圆度计算
- 时间格式化
- 事件描述生成
- HTML 报告生成
- PDF 导出

#### 2. `ReportGenerator.vue` - 报告导出组件

用户界面组件，提供导出按钮和交互逻辑。

#### 3. `ReportCharts.vue` - 报告图表组件（可选）

可视化展示动态参数的变化趋势，包含：

- 亮度变化折线图
- 面积变化折线图
- 周长变化折线图
- 圆度变化折线图

## 文件命名

导出的文件会自动命名为：

```
VAR分析报告_[任务名称]_[时间戳].html
VAR分析报告_[任务名称]_[时间戳].pdf
```

例如：

```
VAR分析报告_视频分析任务_1728745632145.html
VAR分析报告_视频分析任务_1728745632145.pdf
```

## 性能优化建议

### PDF 导出

由于 PDF 生成需要渲染整个 HTML 内容为图片，对于包含大量数据的报告可能需要较长时间：

- 正常情况：3-10 秒
- 大数据量：10-30 秒

导出期间会显示加载状态，请耐心等待。

### HTML 导出

HTML 导出是即时的，不存在性能问题。

## 扩展功能

如需在报告中添加图表可视化，可以在任务详情页面中集成 `ReportCharts` 组件：

```vue
<ReportCharts 
  v-if="task && result"
  :task="task"
  :result="result"
/>
```

这会显示：

- 动态参数的实时变化折线图
- 平均值参考线
- 交互式缩放功能
